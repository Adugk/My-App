    COSC 1100 · Programming
    What is the difference between a compiled and an interpreted language?
    A compiled language (e.g. C) translates source code to machine code before execution. An interpreted language (e.g. Python) executes line-by-line at runtime.

    COSC 1100 · Programming
    What are the three fundamental control structures?
    Sequence (executing in order), Selection (if/else branching), and Iteration (for/while loops).

    MATH 1114 · Mathematics
    What is the difference between decimal, binary, and hexadecimal?
    Decimal (base 10, digits 0–9), Binary (base 2, digits 0–1), Hexadecimal (base 16, digits 0–9 + A–F).

    INFT 1105 · Databases
    What is a Primary Key in a relational database?
    A column (or set of columns) that uniquely identifies each row — it must be unique and not null.

    INFT 1104 · Networking
    What does OSI stand for and how many layers does it have?
    Open Systems Interconnection — 7 layers: Physical, Data Link, Network, Transport, Session, Presentation, Application.
